# AI Productivity Predictor

This is a Machine Learning web application that predicts productivity score based on:

- Hours Studied
- Sleep Hours
- Phone Usage

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas
- Matplotlib

## Features
- ML Model (.pkl file)
- Interactive Dashboard
- Data Visualization
- Live Prediction

## How to Run
streamlit run app.py
